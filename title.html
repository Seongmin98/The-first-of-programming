프로그래밍

프로그래밍(programming)이란

프로그래밍 언어와 프로그램

  1. HTML: 인터넷용 프로그래밍으로 페이지 기술 언어,웹 개발자라면 HTML+CSS는 필수이며 Javascript와 상요작용하는 기초 프로그래밍 프로그램
  2. C언어: 함수로 되어 있는 가장 기초가 되는 프로그래밍 프로그램, 모든 것의 기초를 알지 못하면 모든 것을 할 수 없듯이 반드시 완벽하게 배우고 시작해야하는 컴퓨터 언어 (수동자동차를 다루는 것을 먼저 완벽하게 배워라)
  3. NetBeans: 초보자가 가장 사용하기 편한 Java 개발 환경
  4. Scratch,앤트리: 초등학교 교육용 프로그래밍 개발 툴(미국 메사추세스공과대학/한국 KAIST)
  5. Eclipse: Java대표 개발 환경, 모든 환경에 적용 가능
  6. Microsoft Small Basic: 프로그래밍이라는 것이 어떤 것인지를 공부하는 기본적인 교제 역할,굉장히 심플(한국어 정보 지원 부족)
  7. Unity: 지금 당장 3D게임을 만들 수 있는 프로그래밍(2D도 지원)
  8. Android(Java): 스마트폰 즉 휴대 단말기용 플랫폼(OS)의 기본적인 개발 환경

초보자에게 추천하는 개발 환경

  1. Scratch : 웹 브라우저에 액세스하면 바로 프로그래밍을 시작할 수 있음
  2. Small Basic: 작지만 제대로 된 프로그래밍 언어로 이것으로 프로그래밍 기초를 익힌 후 본격적인 언어를 시작해도 좋음
  3. Xcode: 컴퓨터용 애플리케이션 개발 환경으로 Mac OS X사용자는 Xcode가 필수이지만 Swift로 시작하는 것이 좋기도 함
  4. Eclipse(Jave): 객체지향 언어를 본격적으로 배우는 개발 환경으로서 프로그래머를 직업으로 삼는다면 반드시 배워야할 개발 환경
  5. App Inventor: 스마트폰 앱 개발이 어떤 것인지 시험해 보고 싶다면 좋은 개발 환경
  6. PHP(Notepad++,HTML,CSS,JavaScript): 아마추어가 하기 쉬운 웹 개발 환경
  7. Unity: 게임을 만들고 싶다면 3D 환경을 강력하게 구축한 대표 개발 환경

프로그래머가 주로 취업해서 일하는 분야

  1. 서버관련(Java): 가장 수요가 많고 인기 있는 분야
    인터넷을 사용하는 업무는 느는 한편 일손은 아무리 많아도 부족함
  2. 임베디드 기기 관련(Java/Swift): 개발 중에서 가장 큰 분야
    앞으로 수요가 늘어 인력 부족이 심각해질 분야로써 가전제품,자동차,휴대전화 업무용 기기 등등 이를 제어하는 소프트웨어 기능을 만듦
  3. 업무용 시스템 개발(다양한 프로그래밍): 한 회사의 업무에 맞춰 시스템을 개발하는 분야
    최근에는 네크워크형 시스템도 늘고 있으며 수요가 많음

취업에 도움이 되는 프로그래밍 자격증

  1. 정보처리 기술 관련 시험: 기능사>산업기사>기사>기능장>기술사
  2. 정보처리기사.산업기사 시험: 정보처리에 관한 국가기술자격 중에서 가장 기본이 되는 자격증으로 프로그래머용 시험으로 불림
  3. MOS(Micrisoft Office Specialist): 마이크로소프트가 공인하는 자격중으로 매월 시험이 있으며 오피스 관련 업무시 중요
  4. Java공인 자격증: 일반적인 컴퓨터용 Java인 Jave SE으로 서버 개발자에게 중요
