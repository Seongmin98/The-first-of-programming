프로그래밍에 대해

  1. HTML: 인터넷용 프로그래밍으로 페이지 기술 언어,웹 개발자라면 HTML+CSS는 필수이며 Javascript와 상요작용하는 기초 프로그래밍 프로그램
  2. C언어: 함수로 되어 있는 가장 기초가 되는 프로그래밍 프로그램, 모든 것의 기초를 알지 못하면 모든 것을 할 수 없듯이 반드시 완벽하게 배우고 시작해야하는 컴퓨터 언어 (수동자동차를 다루는 것을 먼저 완벽하게 배워라)
  3. NetBeans: 초보자가 가장 사용하기 편한 Java 개발 환경
  4. Scratch,앤트리: 초등학교 교육용 프로그래밍 개발 툴(미국 메사추세스공과대학/한국 KAIST)
  5. Eclipse: Java대표 개발 환경, 모든 환경에 적용 가능
  6. Microsoft Small Basic: 프로그래밍이라는 것이 어떤 것인지를 공부하는 기본적인 교제 역할,굉장히 심플(한국어 정보 지원 부족)
  7. Unity: 지금 당장 3D게임을 만들 수 있는 프로그래밍(2D도 지원)
  8. Android(Java): 스마트폰 즉 휴대 단말기용 플랫폼(OS)의 기본적인 개발 환경

Netbeans

넷빈즈(NetBeans)는 자바 기반의 사용자 프로그램을 개발하기 위한 플랫폼 또는 자바(Java), 자바스크립트(Javascript), PHP, 그루비(Groovy), C, C++ 등을 개발하기 위한 통합 개발 환경(Integrated Development Environment, IDE)를 말한다. 넷빈즈는 자바로 개발되어 있어 자바 가상 머신(JVM)이 갖추어진 환경이면 사용 가능하며 윈도, 리눅스, 솔라리스(x86/x64, sparc), 맥 OS X의 경우에는 편리한 사용을 위하여 설치 파일이 제공된다. 이클립스와 함께 가장 널리 사용되는 무료 자바 IDE 중 하나이며 2007년 제 17회 졸트 상(Jolt Award)를 수상하였다.(개발환경부문) [1]
